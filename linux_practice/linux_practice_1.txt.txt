1) В своём домашнем каталоге создать подкаталог linux_ex и зайти в него.
перехожу в домашний каталог:    cd ~
создаю подкаталог linux_ex:       mkdir linux_ex
захожу в него:      cd linux_ex
2) Создать файл hello.txt  с текстом “Hello world”:      echo "Hello world" > hello.txt
3) Добавить в этот файл (разделяя пустыми строками)
)свои имя и фамилию:       echo "Сергей Мурашкин" >> hello.txt
)текущую дату:       date >> hello.txt
)имя пользователя:        echo $USER >> hello.txt
зашел через nano  и разделил строки
)содержимое домашнего каталога (нерекурсивно, список имен файлов, включая скрытые):     ls -a /root >> hello.txt
в дальнейшем отделяю пустой строкой через команду:     echo "" >> hello.txt
)содержимое корневого каталога (нерекурсивно):      ls / >> hello.txt
)содержимое каталога /etc:       ls /etc >> hello.txt
)только файлы в каталоге /etc  :      find /etc -maxdepth 1 -type f >> hello.txt
)только подкаталоги в каталоге /etc (нерекурсивно):        find /etc -maxdepth 1 -type d >> hello.txt
)содержимое файла /etc/passwd:         cat /etc/passwd >> hello.txt
)первый столбец файла /etc/passwd (список юзеров):     cut -d: -f1 /etc/passwd >> hello.txt
)содержимое файла /etc/group:       cat /etc/group >> hello.txt
)первый столбец файла /etc/group (список групп)       cut -d: -f1 /etc/group >> hello.txt
4)Вывести hello.txt с прокруткой:     less hello.txt (постранично)    или        cat hello.txt 
5)Разбить hello.txt на файлы размером 1к:      split -b 1k hello.txt  
затем собрать обратно: cat x* > hello.txt
6)Заменить в hello.txt ‘/’ на ‘\’ : cat hello.txt | tr '/' '\' > hello3.txt
7)Скопировать /var/log в текущий каталог (кроме файлов без доступа) с сохранением времени модификации:   cp -rp /var/log .
8)Для полученной копии вывести объем занимаемого места: du -sm * | sort -n
)вывести все файлы, начинающиеся на ‘a’ (не каталоги): find log -maxdepth 1 -type f -name "a*" (глубина в зависимости от поставленной задачи)

Users, groups, access rights

1)Создать трех юзеров (ivanov, petrov, sidorov):     adduser ivanov , adduser petrov , adduser sidorov
 создать каталог /shared  :        mkdir /shared
 дать полный доступ этим двум юзерам (ivanov, petrov): 
 создать группу iv_pet:       addgroup iv_pet
 включить их в группу:     gpasswd -a ivanov iv_pet    ,     gpasswd -a petrov iv_pet
 и изменить доступ:             chmod -R 770 shared
 У остальных не должно быть доступа даже на чтение.
